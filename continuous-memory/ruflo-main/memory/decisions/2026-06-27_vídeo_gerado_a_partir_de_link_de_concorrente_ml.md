# Vídeo gerado a partir de link de concorrente ML

Criamos pipeline que contorna o bloqueio do Mercado Livre usando Zoom como fonte pública de dados. Geramos vídeo base a partir da imagem do produto com texto animado e aplicamos Video Executor para criar 3 variações verticais com abertura animada, CTA e thumbnails.
